Class Mimikatz

(SharpSploit.Credentials.)Mimikatz is a library for executing Mimikatz functions. SharpSploit's implementation uses a PE Loader to execute Mimikatz functions. This is a wrapper class that loads the PE and executes user- specified Mimikatz functions

Remarks

Mimikatz is a tool for playing with credentials in Windows, written by Benjamin Delpy (@gentilkiwi). (Found at https://github.com/gentilkiwi/mimikatz). SharpSploit's PE Loader is adapted from work by Casey Smith (@subtee). (No longer available at original location.) This wrapper class is adapted from Chris Ross (@xorrior)&apos;s implementation. (Found at https://github.com/xorrior/Random-CSharpTools/tree/master/DllLoader/DllLoader)

GetRemoteAccessPolicies(String, String, String, String)

Gets domain computer objects for which remote access policies are applied via GPO.

Declaration
public static bool GetRemoteAccessPolicies(string Domain, string DomainController, string SearchScope, string SearchBase)
Parameters
Type Name Description
System.String Domain

pecifies the domain to use for the query, defaults to the current domain.
System.String DomainController

Specifies an Active Directory server (domain controller) to bind to.
System.String SearchScope

Specifies the scope to search under, Base/OneLevel/Subtree (default of Subtree).
System.String SearchBase

The LDAP source to search through, e.g. /OU=Workstations,DC=domain,DC=local. Useful for OU queries.
Returns
Type Description
System.Boolean

True if execution succeeds, false otherwise.
